# Cloud SOC Threat Detection Platform

A cloud-based Security Operations Center (SOC) simulation built on Azure.

This project demonstrates:

• Cloud infrastructure deployment
• Security monitoring
• Threat detection automation
• Incident alerting
• Attack simulation

Technologies Used:

- Microsoft Azure
- Python
- Azure Monitor
- Log Analytics
- Grafana
- Linux Security Logs

Features:

✔ Detect SSH brute force attacks  
✔ Detect port scans  
✔ Real-time alerting  
✔ Security log analysis  
✔ Incident reporting

Architecture:

Internet
   │
Azure VNet
   ├─ Web Server VM
   ├─ Log Collection VM
   └─ Detection Engine
